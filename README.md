# MERN boilerplate
Boilerplate for pet-projects

## Notice
Minimum supported `Node.js` version is `12.13.0`

## How to use it?
Install modules
```
yarn install
```

Run a project in development mode
```
yarn run dev
```

Run server only
```
yarn run watch:server
```

Run client only
```
yarn run watch:client
```

Build a project
```
yarn run build:prod
```
